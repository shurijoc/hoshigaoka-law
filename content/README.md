## トーンの原則

- 難しい言葉を使わない。法律用語は必ず括弧書きで平易な言い換えを添える
- メリットだけでなく、費用・不利になり得る点・見通しの厳しさも書く
- 「必ず勝てる」「絶対に取り戻せる」等の断定はしない
