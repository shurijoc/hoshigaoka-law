# 取扱分野

星ヶ丘法律事務所は、地域の方から寄せられる一般民事の案件を幅広くお受けしています。企業法務・刑事事件のように高度に専門化された分野は、必要に応じて他事務所をご紹介します。

## 分野一覧

| [離婚](./divorce.md) | 協議離婚 / 調停 / 慰謝料 / 親権 / 養育費 |

## お受けしない・慎重に判断する案件 (現時点の方針)

以下は、単独開業のため十分な体制で対応できない可能性があるものです。ご相談の段階で率直にお伝えします。

- 大規模な企業法務 (M&A / 上場対応 等)
- 重大な刑事事件で長期の身柄拘束を伴うもの
- 医療・特許など高度な専門分野が中心の事件

これらは提携先または他事務所をご紹介します。
